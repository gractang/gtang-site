# gtang-site

Personal website built with Eleventy.

## Getting started

```bash
npm install
npm run start
```

## Build

```bash
npm run build
```

## Notes

- Edit shared layout in `_includes/layouts/base.njk`.
- Update the nav/footer in `_includes/partials`.
- Blog posts live in `writing/`.
- Place your headshot at `assets/headshot.jpg`.
